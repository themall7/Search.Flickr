## Search Flickr Image

### Features

1. Infinite scroll
2. Multiple tag Search
3. Image Popup with all images
4. Ajax loading

### Installation

    git clone https://github.com/themall7/Search.Flickr
    composer update
    setup apache virtual hosting
    test website with web browser
